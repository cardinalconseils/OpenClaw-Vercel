# Phase 1.1 Plan 01: AI Layer — LLM Clients, Orchestrator, Murphy Persona Summary

**One-liner:** Dual LLM client initialization (OpenRouter + Anthropic) with tiered routing orchestrator and Murphy AI persona prompt with FCC/CA SB-1001 disclosure compliance.

## What Was Built

The AI brain of Murphy — four production files that wire together provider SDKs, route by task complexity, define Murphy's persona, and enforce spoken-language constraints for voice calls.

### Files Created

**`src/lib/ai/llm-clients.ts`**
Initializes two LLM clients from environment variables:
- `openRouterClient`: OpenAI SDK pointed at `https://openrouter.ai/api/v1` with `HTTP-Referer` and `X-Title` headers identifying the OpenClaw app.
- `anthropicClient`: Anthropic SDK with standard API key init.
Both use `?? ''` for missing env vars to allow test environments without real keys.

**`src/lib/ai/orchestrator.ts`**
Exports `chat(messages, task)` that routes by task type:
- **Routine** (`greeting`, `intent-capture`, `filler`, `status-update`) → OpenRouter with `google/gemini-2.5-flash-lite`, max 200 tokens (100 for filler)
- **Complex/unknown** (`disambiguation`, `provider-ranking`, `transfer-logic`, `unknown`) → Anthropic with `claude-sonnet-4-5-20251022`, max 500 tokens
Logs routing decision with `[orchestrator] Routing "{task}" to {tier}`.

**`src/lib/ai/prompts/murphy-system.ts`**
Exports `buildMurphySystemPrompt(context?)` with:
- AI identity disclosure in first greeting sentence: "Hi, I'm Murphy — an AI assistant from OpenClaw Service Matchmaker."
- Full 6-stage dispatch pipeline: INTAKE → SEARCH → RANK → DIAL → CONNECT → FOLLOW-UP
- All 4 tool names: `search_providers`, `call_provider`, `transfer_call`, `send_sms`
- Conversation rules (2-turn clarification limit, no dead air, mirror caller language)
- Conditional voice mode via `applyVoiceModifiers()` when `isVoiceCall: true`

**`src/lib/ai/prompts/voice-modifiers.ts`**
Exports `applyVoiceModifiers(basePrompt)` — appends a VOICE MODE block enforcing: no markdown, no bullet points, under 3 sentences, natural speech transitions.
