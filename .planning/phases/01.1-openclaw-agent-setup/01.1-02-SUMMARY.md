## Decisions Made

- **Static TOOLS array:** Tools are known at compile time — no dynamic registry needed. Simpler and fully type-safe.
- **switch/case in executeTool():** Explicit and exhaustive; TypeScript will warn if a handler is missing. Avoids map-lookup indirection.
- **Stub structure:** Each handler logs `[tools:component] STUB —` to distinguish stub execution from real calls during integration testing. Real implementations drop the STUB prefix.
