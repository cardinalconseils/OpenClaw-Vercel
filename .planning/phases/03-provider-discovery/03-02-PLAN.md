    Create `src/lib/voice/narration.ts` following the greeting.ts/filler.ts pattern of hardcoded constants and builder functions.

    **Define a minimal NarrationProvider type** (only the fields narration needs — avoids importing from search.ts which may not exist yet during parallel execution):
    ```typescript
    interface NarrationProvider {
      name: string;
      rating: number;
      distanceKm: number;
    }
    ```

    **buildResultNarration(count, serviceType, location, topProvider, language):**
    - EN: `I found ${count} ${serviceType} providers near ${location}. The top-rated is ${topProvider.name} with ${topProvider.rating} stars — want me to call them?`
    - FR: `J'ai trouve ${count} ${serviceType} pres de ${location}. Le mieux note est ${topProvider.name} avec ${topProvider.rating} etoiles. Dois-je les appeler?`
    - Per CONTEXT.md locked decision: "Summary + top pick pattern"

    **buildNextProviderNarration(provider, language):**
    - EN: `No problem. Next up is ${provider.name} — ${provider.rating} stars, ${provider.distanceKm.toFixed(1)} km away.`
    - FR: `Pas de probleme. L'option suivante est ${provider.name} — ${provider.rating} etoiles, a ${provider.distanceKm.toFixed(1)} km.`
    - Per CONTEXT.md: "If caller declines top pick: Offer next in ranked list"

    **buildNoResultsNarration(serviceType, location, language):**
    - EN: `I couldn't find any matching ${serviceType} providers near ${location}, even with a wider search. I'd suggest trying ${serviceType} ${location} on Google. Sorry I couldn't help more!`
    - FR: `Je n'ai pas trouve de ${serviceType} pres de ${location}, meme avec un rayon plus large. Vous pourriez essayer de chercher ${serviceType} ${location} sur Google. Desole de ne pas pouvoir aider davantage!`
    - Per CONTEXT.md: "No results found: suggest alternatives"

    **buildSearchingFiller(serviceType, location, language):**
    - EN: `Searching for ${serviceType} providers near ${location} now.`
    - FR: `Je cherche des ${serviceType} pres de ${location} maintenant.`
    - This fires alongside the generic filler loop as the initial context-specific filler
