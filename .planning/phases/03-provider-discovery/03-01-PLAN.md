       ```typescript
       export interface Provider {
         name: string;
         phone: string;
         rating: number;
         reviewCount: number;
         address: string;
         distanceKm: number;
         distanceLabel: string;  // "2.3 km"

         placeId: string;
         source: 'google_places' | 'web';
       }
       ```

       **SearchProvidersParams** — keep same shape (service_type, location, urgency) plus add optional `callControlId` for CallState update.

       **SearchProvidersResult:**
       ```typescript
       export interface SearchProvidersResult {
         providers: Provider[];
         source: string;
         count: number;
         narrationText?: string;
       }
       ```

       **geocodeLocation(location, apiKey):**
       - GET `https://maps.googleapis.com/maps/api/geocode/json?address=${encodeURIComponent(location)}&key=${apiKey}`
       - Guard: if `!json.results?.length` throw `Error('[search] Could not geocode location: "${location}"')`
       - Return `{ lat, lng }` from `json.results[0].geometry.location`

       **haversineKm(lat1, lng1, lat2, lng2):**
       - Standard haversine formula with R = 6371 km
       - Returns distance in kilometers as a number

       **PLACES_FIELD_MASK** constant:
       ```
       places.id,places.displayName,places.formattedAddress,places.location,places.rating,places.userRatingCount,places.nationalPhoneNumber,places.currentOpeningHours
       ```

       **callPlacesApi(textQuery, lat, lng, radiusMeters, urgency, apiKey, languageCode?):**
       - POST to `https://places.googleapis.com/v1/places:searchText`
       - Headers: `Content-Type: application/json`, `X-Goog-Api-Key: apiKey`, `X-Goog-FieldMask: PLACES_FIELD_MASK`
       - Body: `{ textQuery, pageSize: 10, rankPreference: urgency === 'emergency' ? 'DISTANCE' : 'RELEVANCE', locationBias: { circle: { center: { latitude: lat, longitude: lng }, radius: radiusMeters } }, languageCode: languageCode ?? 'en' }`
       - Return `response.places ?? []` (guard against missing key on zero results)
       - Filter out results with no `nationalPhoneNumber` (useless for Phase 4)

       **scoreProvider(provider, urgency):**
       - ratingScore = (provider.rating / 5) * 100
       - proximityScore = Math.max(0, 100 - provider.distanceKm * 5) (100 at 0km, 0 at 20km)
       - reviewScore = Math.min(100, Math.log10(provider.reviewCount + 1) * 50)
       - openScore = provider.isOpenNow ? 20 : 0
       - Normal: ratingScore * 0.40 + proximityScore * 0.35 + reviewScore * 0.15 + openScore * 0.5
       - Emergency: proximityScore * 0.40 + openScore * 1.5 + ratingScore * 0.20 + reviewScore * 0.10

       **mapPlaceToProvider(place, callerLat, callerLng):**
       - Map Google Places response fields to Provider interface
       - Compute distanceKm via haversineKm
       - Set distanceLabel = `${distanceKm.toFixed(1)} km`
       - Set isOpenNow from `place.currentOpeningHours?.openNow`
       - Extract openingHoursText from weekdayDescriptions[0] or undefined
       - Set source = 'google_places'

       **searchProviders(params):**
       - Read `GOOGLE_MAPS_API_KEY` from `process.env`
       - Call geocodeLocation(params.location, apiKey)
       - Set initial radius 5000m, call callPlacesApi
       - If results < 3, expand radius to 25000m, call again
       - Map results to Provider[] via mapPlaceToProvider
       - Score and sort descending via scoreProvider
       - If params.callControlId provided, call updateCall(callControlId, { providers: ranked, currentProviderIndex: 0, stage: 'searching' })
       - Return { providers: ranked, source: 'google_places', count: ranked.length }
       - Log with `[tools:search]` prefix throughout
